# extended-yank

This emacs package provides the ability to read the rich format of the clipboard so that users can paste more information into emacs. Inspired by [[https://github.com/citrus-lemon/emacs-macos-clipboard]]

## Dependencies

### macOS

This package relies on shell calls to osascript and pandoc. osascript should be a native utility on your macOS device. To get pandoc:

`brew install pandoc`


## Installation

git clone this directory into your emacs.d directory, then

`(require extended-yank)` in your init file.
